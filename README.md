# Research Flow

[Live GitHub Pages site](https://baditaflorin.github.io/research-flow/)

Private browser research workspace for clustering papers, finding gaps, drafting outlines, and exporting cited Word/LaTeX.

Research Flow is a local-first alternative to a pile of paid tools for the "I'm researching X" workflow. Drop papers into one URL, keep the files in your browser, and turn the reading pile into a research map with clusters, contradictions, gaps, an editable outline, citations, and exports.

## Quickstart

```sh
npm install
make install-hooks
make dev
```
