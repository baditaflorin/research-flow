# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while still being precise about risks and defects.
- Keep disagreements technical and grounded in evidence.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without explicit permission.
- Submitting malware, credential leaks, or knowingly unsafe code.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors that violate this code of conduct.
